#  World Data Analysis using SQL, Pandas & Matplotlib

##  Project Overview

This project focuses on analyzing global data including countries, cities, and languages using SQL and Python.

##  Technologies Used

* MySQL (Database)
* Python (Pandas)
* Matplotlib (Data Visualization)
* SQLAlchemy (Database Connection)

##  Key Analysis Performed

* Top countries and cities by population
* Population distribution across continents
* Language usage trends
* Economic analysis using GNP
* Relationship between population and life expectancy

##  Visualizations

* Bar charts (population, cities, languages, GNP)
* Pie chart (population distribution)
* Scatter plot (life expectancy vs population)

##  Key Insights

* Asia dominates global population due to countries like China and India
* Urban population is concentrated in major metropolitan cities
* Economic power is concentrated in a few developed countries
* No direct relationship between population and life expectancy

##  Learning Outcomes

* Writing efficient SQL queries (SELECT, JOIN, GROUP BY)
* Performing data analysis using Pandas
* Creating meaningful visualizations using Matplotlib
* Extracting real-world insights from data


⭐ This project demonstrates beginner-level data analysis skills using real-world datasets.
